# Cybersecurity # Networking Quiz Program

This is a Python quiz program (terminal-based) that tests users on networking and cybersecurity concepts.

# Features
- Randomized question order
- Score tracking
- Multiple choice questions
- Case-insensitive answer checking

# Technologies Used
- Python
- Visual Studio Code
- GitHub

- Nested data structures
- Functions
- Loops & Nested Loops
- Randomization
- User input validation
